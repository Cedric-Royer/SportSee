(async, constant) aggregateUserData

Description:

  • Agrège toutes les données utilisateur nécessaires en effectuant des appels API ou en utilisant des données simulées. La fonction utilise Promise.allSettled pour gérer indépendamment les erreurs de chaque appel API. En cas d'échec d'un appel, des valeurs par défaut sont utilisées pour éviter l'échec global.
